About.

Het ontwikkelen en optimaliseren van applicaties is mijn passie. Ik ben dynamisch, positief, communicatief, oplossings- en resultaatgericht. Mijn ervaring ligt vooral op het gebied van JavaScript (ES5/ES6), CoffeeScript, Ruby on Rails en SCSS/CSS. Mijn ambitie is om mijzelf verder te ontwikkelen tot Full-stack JavaScript Developer.

Sinds begin dit jaar (2018) ben ik begonnen bij Competa in Rijswijk als Fullstack Javascript Developer. Hier werk ik, naast mijn persoonlijke project mee aan een MERN (MongoDB, Express, React en Node.js) project, dit is precies de stack die ik wilde leren. Echter kan ik niet wachten om AngularJS en Vue.js toe te voegen aan mijn skill set, deze staan op de agenda voor de komende jaren.

SKILLS
